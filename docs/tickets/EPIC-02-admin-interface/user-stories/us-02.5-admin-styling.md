## Enqueue Location

Add to `includes/class-gap-meta-boxes.php`:

```php
/**
 * Enqueue admin CSS
 */
public function enqueue_admin_styles($hook) {
    // Only enqueue on tracking_script edit pages
    if ('post.php' !== $hook && 'post-new.php' !== $hook) {
        return;
    }

    global $post_type;
    if ('tracking_script' !== $post_type) {
        return;
    }

    // Enqueue admin CSS
    wp_enqueue_style(
        'gap-admin-css',
        GAP_PLUGIN_URL . 'assets/css/admin.css',
        array(),
        GAP_PLUGIN_VERSION
    );
}

// Hook to admin_enqueue_scripts
add_action('admin_enqueue_scripts', array($this, 'enqueue_admin_styles'), 10, 1);
```
